よく分からんな〜って感じで、いつもなら
そっと目を閉じ記憶を消し、普段の日常を送っていたが今回はせっかく決めたしワクワクしてたからやってみようと思う。

hono と bun っていうものを使って個人サイト作ろうと決めた。何でこれにしたのかは、hono と bun の組み合わせは
高速っていう感じのが書いてあった気がしたので、高速という魔力に引っ張られて決めました。
HTML と CSS は大学のサークルでかじったことがあるので問題ないとして、フレームワークってのはサイトを作る上で便利ってのは
わかったんだけど、うまいちそれ以上のことが頭に入ってこなかった。

やっぱり自分はコツコツ勉強より、必要になってから勉強!!みたいなタイプなので今回も作りつつその都度必要になったら調べて
行こうと思う。調べていく中で、hono と bun は比較的新しいツールっぽいことがわかってきた。なのでほぼほぼ先人がいないので
ちょっときつくて、個人サイト制作に無理ゲー感が頭の中をじわじわと汚染してきた。まぁ決めたから無理くりでもやるんだけどね(๑^᎑^๑)

大学のテスト期間を挟みながら 3 週間ぐらい、隙間時間を活用しながらやっとのことで大体の構造やデザインができた。最後ら辺は夏休みに
入ったので一気に進んだ。あとは無料であげれると噂の Cloudflare Pages にあげれば完成(ง ᵕωᵕ)ว ﾙﾝﾙﾝ ♪ なんて思ってたのに、
bun がロックされてる的なエラーに対応できず、無事死亡＼(^o^)／

仕方ないので、フレームワークから全て変えることにした。今度は歴史がありそうな Next.js っていうフレームワークを使うことにした。
なぜこのフレームワークにしたのかというと、サイトの path がフォルダ名で勝手に決まってくれるっていうのがいいなぁ、
感じたので Next.js で一から作ることにする。

まだまだ、ブログ開設への道のりは遠いみたいだ(;´д ｀)ﾄﾎﾎ…

皆さんは新しいことに挑戦するとき、まず網羅的に知ってから進んでいく派? それともとりあえず突っ走って穴にはまっては抜け出す
みたいな感じで進んでいく派ですか?それぞれにメリット・デメリットあると思いますが、私は突っ走ることをお勧めします。
